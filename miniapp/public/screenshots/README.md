# Картинки для лендинга и соц. превью

## Скриншоты приложения (для секций лендинга)

LandingScreen.tsx ссылается на следующие файлы:

| Файл               | Где используется                       | Что должно быть на скриншоте                                  |
|--------------------|----------------------------------------|---------------------------------------------------------------|
| `main.png`         | Hero (правая колонка)                  | Главный экран — «Что тренируем сегодня?» с 4 режимами + stats |
| `speaking-fix.png` | Killer-feature (скриншот разбора)      | Speaking с «Почему так?» — желтое выделение исправления       |
| `grammar.png`      | Галерея «Как это выглядит» (колонка 1) | Урок грамматики (например «Местоимения I/my, you/your»)        |
| `listening.png`    | Галерея «Как это выглядит» (колонка 2) | Listening — плеер с подкастом + словарь, тап на слово         |
| `progress.png`     | Галерея «Как это выглядит» (колонка 3) | Мой прогресс — streak, минуты, медали                         |

**Рекомендации:**
- Формат: **PNG** (для чёткости UI) или WebP, ширина 600–900px.
- iPhone screenshot 1170x2532 → пересохрани в ~800px ширины, иначе лендинг будет тяжёлым.
- Без обрезки статус-бара устройства — пусть скрин выглядит «как у юзера на телефоне».
- Если каких-то файлов не будет — рамки на лендинге останутся пустыми (alt-text покажет название), вёрстка не сломается.

## og-cover.png (превью при шаринге в соц. сетях)

Положи отдельную картинку **`miniapp/public/og-cover.png`** (не в эту папку, а на уровень выше):

- **Размер: 1200 × 630 px** (стандарт Open Graph для Facebook/VK/Telegram/Twitter).
- Формат: **PNG** или **JPEG**, до 1 МБ.
- На картинке: крупно название «English Tutor», слоган типа «Заговори, а не молчи», и желательно скриншот или иллюстрация (не пустой фон). Минимум текста — превью маленькое.
- Если файла нет — превью при шаринге будет без картинки (только title + description из meta-тегов). Не критично, но менее привлекательно.

Проверить превью после деплоя можно тут:
- Telegram — отправить ссылку в любой чат.
- VK — https://vk.com/dev/pages.clearCache (после первого шаринга кеш ВК-парсера живёт долго).
- Facebook/общий валидатор — https://www.opengraph.xyz/url/https%3A%2F%2Fenglishbot.krichigindocs.ru%2F
