# Скриншоты для лендинга

LandingScreen.tsx ссылается на следующие файлы — положи сюда свои скриншоты приложения:

| Файл               | Где используется                       | Что должно быть на скриншоте                                  |
|--------------------|----------------------------------------|---------------------------------------------------------------|
| `main.png`         | Hero (правая колонка)                  | Главный экран — «Что тренируем сегодня?» с 4 режимами + stats |
| `speaking-fix.png` | Killer-feature (скриншот разбора)      | Speaking с «Почему так?» — желтое выделение исправления       |
| `grammar.png`      | Галерея «Как это выглядит» (колонка 1) | Урок грамматики (например «Местоимения I/my, you/your»)        |
| `listening.png`    | Галерея «Как это выглядит» (колонка 2) | Listening — плеер с подкастом + словарь, тап на слово         |
| `progress.png`     | Галерея «Как это выглядит» (колонка 3) | Мой прогресс — streak, минуты, медали                         |

**Рекомендации:**
- Формат: **PNG** (для чёткости UI) или WebP, ширина 600–900px.
- Если у тебя iPhone screenshot 1170x2532 — пересохрани с уменьшением до ~800px ширины, иначе лендинг будет тяжёлым.
- Без обрезки статус-бара устройства — пусть скрин выглядит «как у юзера на телефоне».
- Если каких-то файлов не будет — рамки на лендинге останутся пустыми (alt-text покажет название), но это не критично; вёрстка не сломается.
